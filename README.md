# Food Delivery Operational Analysis

## Overview
Exploratory data analysis project focused on understanding delivery performance and customer behavior.

## Objective
Analyze patterns in delivery time, ratings, and cuisine demand to generate operational insights.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn

## Key Insights
- Higher demand observed during weekends.
- Large percentage of unrated orders.
- Delivery time variability across conditions.

## Recommendations
- Improve customer feedback collection.
- Optimize resource allocation during peak demand.

## Engineering Perspective
Analysis performed with focus on operational performance and data-driven decision making.
